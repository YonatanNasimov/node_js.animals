Documentation of my animals API

Hello I'm Yonatan Nasimov and this Documentation of my animals API I've written.

animals Routes

Get information about animals via a full API

Method URL Description
Get /animals?page=number Get a list of all the animals , present 10 results by page number
Get /animals?perPage=number Get X animals in each page
Get /animals?reverse=sort-key Reverse by sort key: yes/no
Get /animals?sort=key-word From which value to sort by
Get /animals/price?min=number&max=number Price range
Get /animals/category/:category-name Get all animals from the same category
Get /animals/search?s=keyword Search products by name or info

animals Actions

Add, remove and edit allowed only for registered users

Method URL Body Description
Post /animals
  1. name (required)
  2. type (required)
  3. info (required)
  4. category (required)
  5. img_url
  6. price (required)
 Add a new animal Must to be connected
Edit /animals/:id-key
  1. name (required)
  2. type (required)
  3. info (required)
  4. category (required)
  5. img_url
  6. price (required)
 Edit a new animal Must to be connected
Delete /animals/:id-key Delete a new animal Must to be connected
